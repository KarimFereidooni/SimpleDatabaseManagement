# Product Overview

Simple Database Management is a Windows Forms desktop application for managing SQL Server databases. It provides a GUI for common database administration tasks including:

- Connect to SQL Server instances (Windows or SQL authentication)
- View and manage databases
- Create, attach, detach, and delete databases
- Backup and restore databases
- Execute SQL queries
- View database properties
- Install database updates from SQL script files

The application targets database administrators and developers who need a lightweight tool for routine SQL Server database operations.
